Iphone 15 clone

This project is a front-end clone of Apple’s iPhone 15 product page. It’s built with React, styled using Tailwind CSS, and features animations powered by GSAP (GreenSock Animation Platform) and Three.js 3D effects. The goal is to replicate the smooth scroll-based animations and polished, responsive design of Apple’s original site.
